# ElderCare Management Tool

ElderCare is a **web-based management tool** designed to help efficiently run and organize old age homes.  
It provides a simple, intuitive interface for staff and administrators to manage residents, staff, and daily operations with ease.

- **Activity Scheduling:** Plan daily activities for residents.
- **Donation Tracking:** Keep a record of donations and contributors.
- **Simple and Responsive UI:** Built for non-technical users.

The tool emphasizes accessibility, usability, and clarity, making it ideal for elder care environments.
